A1 Vue App

Setup

Backend: 

cd backend
npm install
npm run dev

Backend runs at: http://localhost:3000

Frontend

cd frontend
npm install
npm run dev

Frontend runs at: http://localhost:5173

Pages:
- `/` - Home
- `/about` - About
- `/contact` - Contact form
- `/leaderboard` - Leaderboard table
